# MkExGrAg
Projet Web ET4 Polytech Paris-Sud - Make the Extranet Great Again


Voici les différentes étapes pour lancer notre projet:
- Tout d'abord, installer Visual Studio Community 2017 avec les options suivantes:
	- Développement web et ASP.NET
	- Stockage et traitement de données
	- Développement .NET Desktop
- Une fois que VS est intallé et lancé, aller dans la fenêtre "Team explorer", se connecter.
- Ensuite cloner le repository.
- Puis, Ouvrir la solution (apparue un peu plus bas)
- Et enfin cliquer sur la flèche verte au-dessus (ou déboguer, exécuter avec débogage), alors un onglet apparaîtra dans votre navigateur Web par defaut comprenant la page d'accueil de notre site.
